# Gnosis

RPG de turnos desenvolvido para navegador como projeto de Programação Web.

O jogo foi construído utilizando **HTML, CSS e JavaScript puro**, com foco em lógica de combate por turnos, organização de código orientada a objetos e interação direta com o DOM. A ambientação segue uma temática **dark fantasy**, com trilha sonora e efeitos sonoros integrados.

## Funcionalidades

- Jogo 100% executado no navegador (sem backend)
- Seleção de classe:
  - Guerreiro
  - Mago
  - Arqueiro
- Sistema de combate por turnos
- Ataques básicos e ataques especiais com consumo de mana
- Inimigos com dificuldade progressiva:
  - Goblin
  - Orc
  - Dragão
- Barras visuais de vida (HP) e mana
- Feedback visual e sonoro durante as batalhas
- Modais de vitória e derrota
- Controle de áudio global

## Tecnologias Utilizadas

- **HTML5** – Estrutura da aplicação
- **CSS3** – Estilização, layout e ambientação visual
- **JavaScript (ES6+)** – Lógica do jogo, classes, controle de estado e interação

## Objetivo do Projeto

Este projeto teve como objetivo aplicar conceitos fundamentais de:
- Programação orientada a objetos em JavaScript
- Manipulação do DOM
- Controle de fluxo e estados
- Estruturação de um jogo funcional no ambiente web
